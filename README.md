# INS_IEP

## Installation
* Download the package on your PC.
* Download the EasySpin package - details below.
* Open MATLAB.
* Add both the INS_IEP/INS_IEP and EasySpin/easyspin directories to your MATLAB path, e.g. by `addpath(genpath('INS_IEP/INS_IEP'))` and `addpath(genpath('EasySpin/easyspin'))`.

### Verify Installation

To verify that instalation was successful navigate to the directory INS_IEP/Tests and run the tests file `iitest`.

### Dependencies 
Note that this package requires [`easyspin`](https://easyspin.org) to be installed and added to the MATLAB path. This can be downladed at https://easyspin.org.


## Repository Organization

The INS_IEP repository is organized as follows:

INS_IEP/      source code for the package
Examples/        collection of INS_IEP example scripts
Paper/        source code for the JOSS paper (under review)
Tests/        unit tests for INS_IEP
